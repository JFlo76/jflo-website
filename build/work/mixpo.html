Mixpo

Mixpo provides a web-based application that enables publishers to create and manage interactive advertising campaigns.

Mixpo's original software product was built on Flash, and it was time to bring it into a modern tech stack. As an employee at Mixpo, I helped to build the UI of the application and pattern library using AngularJS and Bootstrap.

Working closely with designers and project managers, I also assumed an active role as UX advocate, suggesting patterns to help reduce the chance of user error and provide meaningful feedback.

Toolkit: NPM, Grunt, Git, AngularJS, Twitter Bootstrap

Project Status: Live (via paid service only)
